This project shows how you can integrate the fun wallet with a angular app.

Full documentation is available on how you can integrate is [here](https://funfair-tech.github.io/fun-wallet-docs/guide/web-sdk/installing-sdk.html)

## Install

```bash
$ npm install
$ npm start
```

This will load the app on http://localhost:3001
The page will reload if you make edits.<br />
You will also see any lint errors in the console.

## Live demo

To see it in action please go to - https://funfair-tech.github.io/wallet-react-integration-sample-js (this is the react deployed codebase but shows the wallet in action)
